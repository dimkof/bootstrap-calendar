Example

Here is an example use-case of this plugin. It adds notes to any date on the calendar. Currently, this example saves data in the localStorage.

Markup

Just by adding the following markup, you will get the calendar rendered like the one above. It internally leverages Bootstrap's Carousel plugin for the transition and comes with a default template. However, you would be able to configure the templates based on your needs. 

<div data-toggle="calendar"></div>
